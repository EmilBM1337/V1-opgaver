Om CSS

Hvad er en selector med egne ord?

Det er det som skal ændres.

Hvad er en property med egne ord?

Det er den egenskab der skal ændres på.

Hvad er en value med egne ord?

Det er den værdi som egenskaben skal have.

Oversigt over simple selectors
Selector: Eksempel: Forklaring:
Element p Vælger alle elementer af denne type, f.eks. alle paragraphs
Id #someId Bruges til a identificere et enkelt element
Class .question Bruges til at definere en grupper HTML elementer for at anvende en unik style
Universal * Bruges til at vælge alle typer elementer
Grouping table, tr, td, th Bruges til at samle selectors der skal have den samme style
