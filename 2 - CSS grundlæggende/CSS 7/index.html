Skriv med egne ord hvad display property påvirker? Den påvirker et element.

Hvad gør display: block? Den Viser et element som et block element (som p). Den starter på en ny linje og fylder hele bredden.

Hvad gør display: inline? Den Viser et element som et inline element (f.eks. span). hight og width properties har ingen effekt.

Hvilke fordele er der ved display: inline-block? Den viser et element som en blockbeholder på inline niveau. Selve elementet er formateret som et inline element, men du kan anvende height og width.
